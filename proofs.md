# Proofs of this Game

## Tutorial

1. p → q → p
2. p → ((p → q) → q)
3. (q → r) → ((p → q) → (p → r))  syllogism
4. q → (p → (p ∧ q))
5. q → (p → (r → p ∧ (q ∧ r)))
6. (q ∧ p) → q
7. (p ∧ q) → (q ∧ p)
8. p → (p ∨ q)
9. q → (p ∨ (q ∨ r))
10. (p ∨ q) → ((p → q) → q)
11. (p ∨ q) → (q ∨ p)
12. (f → p)
13. ¬p → (p → q)
14. p → ¬¬p
